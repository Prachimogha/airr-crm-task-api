# AIRR CRM Task API

## 🚀 Tech Stack
- Django
- Django REST Framework
- SQLite
- GitHub Actions (CI)

## ⚙️ Setup

1. Clone repo
2. Create virtual environment
3. Install requirements:
   pip install -r requirements.txt
4. Run migrations:
   python manage.py migrate
5. Run server:
   python manage.py runserver

## 🧪 Run Tests
python manage.py test
